# SRPI-Game-Project

一个基于 Godot + C# 开发的原创乐队题材 AVG叙事 / 策略 / Meta游戏项目。

## 版本
Alpha 0.01.20260309

## 技术栈
- Godot（Godot_v4.6.1-stable_mono_win64）
- C#（.NET SDK 9.0）
- Git / GitHub
- SQLite

## 当前状态
- 项目原型开发中
- 已搭建基础场景与对话系统

## 目录结构
- Scripts: 脚本
- Dialogues: 对话内容
- Resource: 资源文件
- Scenes: 场景相关内容
- Theme：美术UI的相关主题

## 技术细节
- 利用SQLite搭建对话数据库，导出CSV文件再转换为Json文件到游戏数据内，实现对话系统数据的搭建
